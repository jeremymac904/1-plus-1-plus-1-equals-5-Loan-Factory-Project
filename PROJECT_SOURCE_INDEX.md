## How to update this index

When adding a new file:

1. Add a row to the appropriate section
2. Set Path, Purpose, Owner, Status
3. If it's a template, name with `TEMPLATE_` prefix
4. If it replaces an existing file, mark the old one Archived and move it to `ARCHIVE/`

When moving a file:

1. Update the path in this index
2. Update any cross-references in other docs that point to it

When archiving a file:

1. Move the file to `ARCHIVE/`
2. Change Status to Archived in this index
3. Add a short note about why in the file itself
